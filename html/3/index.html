This one is for you typophiles out there. For a long time web-typography was pretty borked, this changed with wide-spread support for @font-face. But still you couldn't really properly use the fonts you had to their full extend: you couldn't use small-caps, ligatures, old-style figures and swashes. With the css property font-feature-settings you now can.

Here are some examples

How does it work?

The font-feature-settings CSS property gives us access to popular OpenType features. This gives us the power to enable or disable certain font features whenever we need it. To enable old-style numerals we set font-feature-settings: "onum". You can combine multiple features by appending them, so if we wanted to use old-style numerals like in the table above we'd use: font-feature-settings: "onum", "lnum".
